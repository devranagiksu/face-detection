# 👤 Face Detection with OpenCV

## 📌 Description
This project demonstrates **face detection using OpenCV** and Haar Cascade classifiers.

It detects human faces in an image and draws rectangles around them.

---

## 🎯 Features
- Detect faces in images  
- Uses pre-trained Haar cascade model  
- Simple and fast implementation  

---

## 🛠️ Requirements
- Python 3.x  
- OpenCV
